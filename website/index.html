Installation

Up and running in seconds

Requires Rust 1.75+ and a Cerebras API key.

1

Clone the repository

git clone https://github.com/yourname/otask && cd otask
2

Add your Exa API key

echo "EXA_API_KEY=your_key_here" > .env
3

Build and run

cargo build --release && ./target/release/otask
4

Connect your Cerebras key and start chatting

/connect cerebras YOUR_CEREBRAS_KEY
p               # switch to plan mode
i               # start typing
